Step 1: Download server.py

Step 2: Shift + right click the file server.py

Step 3: Open in Powershell

Step 4: Type in "python (script name)" <-- server.py

Step 5: Run

Step 6: Open a web browser and type in the ServerPort number in the code into the search/url bar <-- (Localhost:12000)

Step 7: Execute and check PowerShell to see the web server.
